# Attachmi
Attachmi is a desktop app for structured document storage, designed for the modern knowledge worker.


## Development

Running the app:
```bash
cd angular && npm start             # Start Angular dev server (terminal 1)
cd electron && npm run dev          # Start Electron with auto-restart (terminal 2)
```

```
Tech Stack:
- Node.js 24 (backend)
- Electron
- Angular
- Sqlite
