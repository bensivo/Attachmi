# Test Suite: Basic Functionality

### TC - Add Attachment
Steps: 
- Click "Add Attachment"
- Fill out the form
- Submit
- Select the new attachment from the list

### TC - Open File
Steps
- Select an attachment
- Click "Open File"
- Verify it opens in your browser's default app for that filetype

### TC - Delete attachment
Steps:
- Select an attachment
- Click "Delete"

### TC - Edit attachment name, date, description
Steps: 
- Select an attachment
- For each of the fields:
    - Edit it
    - Go to another attachment, and back
    - Verify it's still there. 

### TC - Search by name
Steps:
- Type in the name of any attachment in teh searchbar
- Verify it appeasr in the list, everything not matching removed
- Verify it works case-insensitive

### TC - Search by description
Steps:
- Type some text from an attachment's description
- Verify it appeasr in the list, everything not matching removed
